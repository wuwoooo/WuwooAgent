编译（需已安装 JDK 17+ 与 Android SDK，并配置 sdk.dir）：

1. 复制 local.properties.example 为 local.properties，将 sdk.dir 改为本机 SDK 路径。
2. 在 agent-ime 目录执行：./gradlew :app:assembleDebug
3. APK：app/build/outputs/apk/debug/app-debug.apk

安装：adb install -r app/build/outputs/apk/debug/app-debug.apk

未安装 Android Studio 时：仅安装 Command line tools + sdkmanager 安装 platform-tools、build-tools、platforms;android-34 即可。
